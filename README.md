# 👁️ CyberAscii Vision

A futuristic, real-time camera-to-ASCII converter with integrated AI scene analysis. Experience the world through the eyes of a cyberpunk security system.

## ✨ Features

- **Live ASCII Feed**: High-performance real-time conversion of your camera stream into ASCII characters.
- **🤖 AI Neural Link**: Integrated Google Gemini AI that analyzes your "visual data" and provides robotic threat assessments and identifiers.
- **🎨 Multiple Vision Modes**: Choose between *Matrix Green*, *Retro Amber*, *Classic BW*, and *Chrome Color*.
- **⚙️ Dynamic HUD**: Adjustable font size, gain (brightness), and dynamic range (contrast) via a collapsible system configuration panel.
- **📱 Mobile Optimized**: Fully responsive interface with a dedicated camera flip feature for front/back sensor toggling.
- **🔊 Sci-Fi Soundscapes**: Atmospheric ambient hum and interactive UI sound effects for full immersion.

## 🚀 Quick Start

### Prerequisites
- Node.js (v20+)
- A Google Gemini API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/GodfatherTHR/Ascii-art.git
   cd Ascii-art
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Launch Application**
   ```bash
   npm run dev
   ```
